# failed\_when and until

Q: Does `failed_when: false` override the result of `until`, even if the retry
   limit is reached?

A: Yes.

```
PLAY [localhost] *******************************************************************************************************************************************************************************************

TASK [Gathering Facts] *************************************************************************************************************************************************************************************
ok: [localhost]

TASK [command] *********************************************************************************************************************************************************************************************
FAILED - RETRYING: command (2 retries left).
FAILED - RETRYING: command (1 retries left).
changed: [localhost]

PLAY RECAP *************************************************************************************************************************************************************************************************
localhost                  : ok=2    changed=1    unreachable=0    failed=0   
```
